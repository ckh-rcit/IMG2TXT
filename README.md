# IMG2TXT

Describe images using local vision models running on Ollama. Drag and drop an image, select a model, and get a description generated by your local LLM.

## Features

- Drag-and-drop or paste image input
- Auto-discovers all locally available Ollama models, sorted with vision-capable models first
- Prompt modes: Default description, OCR (text extraction), FLUX.1-dev prompt generation, FLUX.2-dev prompt generation
- Streaming response displayed token-by-token
- Character count and copy-to-clipboard

## Requirements

- [Bun](https://bun.sh) runtime
- [Ollama](https://ollama.ai) running locally with at least one vision model pulled (e.g. `ollama pull llava`)

## Getting started

```bash
bun install
bun dev
```

Opens at `http://localhost:5173`. The Vite dev server proxies `/api/*` requests to Ollama at `localhost:11434`.

## Build

```bash
bun run build
```

Output goes to `dist/`. Serve with any static file server.

## Stack

- React 19, TypeScript 6
- Vite 8, Tailwind CSS v4
- shadcn/ui components
- Bun runtime
